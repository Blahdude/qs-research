Midjourney Workflow

Last updated: Tuesday July 9, 2024

AI Research Tools Use Cases:

First Step: If you don’t already have an account or still need to set up your account, follow the instructions on this website: https://zapier.com/blog/how-to-use-midjourney/

  • Using images as inspiration (recommended)
    • Making Midjourney prompts can be tedious, frustrating, and complicated because sometimes the outputs just aren’t anything like the result you want. This feature “/describe” streamlines the prompt creation process through using images as inspiration. Midjourney will take an image, write a prompt that perfectly describes it, and use the prompt to create images like the one you showed them.
    • Steps:
      • Type “/describe” within the text input for Midjourney Bot
      • Once you enter “/describe”, an options tab will pop up. Click “image”.
      • Now, drag and drop an image you want Midjourney to use as inspiration.
      • Once you see your image appear in the chat box, press enter.
      • You should now see a series of prompts, with numbers corresponding to each prompt. If you have a prompt you like best, click on the number for the corresponding prompt. However, it is recommended to click “Imagine all” in order to get a wide variety of results.
      • Now, if you want to customize further or export an image, press ‘U’ + # of the image you want customized/exported. For example, I want to customize image 4, so I'm pressing “U4.” OR if you like a particular image, but want to generate more images like it, press ‘V’ + ‘# of the image’ and you will get 4 more different versions similar to that image.
      • If you pressed ‘U’, you will be given a series of new buttons. If you want to edit further, feel free to explore these buttons and edit the photo as you wish.
      • Congratulations! Now you have some photos from Midjourney that can be used freely for commercial use!
